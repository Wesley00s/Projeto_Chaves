-Criar uma estrutura de dados: 
Você pode criar uma estrutura para armazenar as informações de cada registro de chave, como nome da pessoa, horário, sala e se é uma entrega ou devolução.

-Implementar a funcionalidade de registro de entrega:
 Crie uma função ou método que permita registrar a entrega de uma chave, solicitando e armazenando o nome da pessoa, o horário e a sala.

-Implementar a funcionalidade de registro de devolução:
 Da mesma forma, crie outra função ou método para registrar a devolução da chave, solicitando e armazenando o nome da pessoa, o horário e a sala.

-Adicionar uma identificação visual:
 Você pode adicionar uma representação visual das chaves (como um array de booleanos), onde cada posição do array representa uma chave e seu valor (true/false) 
indica se está disponível ou não. Isso pode ser útil para visualizar rapidamente quais chaves estão emprestadas.

-Implementar uma interface de usuário: (aqui no caso viria primeiro que os passos anteriores)
 Crie uma interface simples para interagir com o programa, permitindo que o usuário escolha entre registrar a entrega ou devolução de uma chave e fornecendo os 
campos necessários para cada ação.

-Adicionar funcionalidades extras: 
Você também pode adicionar funcionalidades extras, como a possibilidade de cadastrar novas chaves ou pessoas, gerar relatórios de uso das chaves, entre outros.
